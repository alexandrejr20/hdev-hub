Desenvolvedor Front-End (voluntário)

Descrição

A HDEV está em busca de desenvolvedores Front-End voluntários em início de carreira para ajudar no desenvolvimento de interfaces web.

Você será responsável por criar e implementar designs de páginas web, garantindo que sejam responsivas e funcionais.

🎯Responsabilidades

Desenvolver interfaces web utilizando HTML, CSS e JavaScript.

Utilizar Git e GitHub para versionamento e envio de código.

Corrigir bugs simples relacionados ao layout ou funcionamento da interface.

Participar de reuniões de acompanhamento do projeto.

🌱O que ganha sendo desenvolvedor da HDEV?

Experiência profissional na área como desenvolvedor Front-End.

Networking com outros profissionais da área.

Contribuição direta para a formação de novos desenvolvedores.

Fortalecimento do seu perfil profissional.

🧠Requisitos

Estar cursando alguma faculdade relacionada na área.

Boa comunicação e vontade de aprender.

Proatividade e capacidade de trabalhar em equipe.

💙Diferenciais

Inglês intermediário ou avançado.

Experiência com alguma linguagem de programação.

Experiência com frameworks.
